# Actividad: Cola de turnos y pila de historial

## Objetivo
Implementar una cola FIFO y una pila LIFO en Java.

## Instrucciones
1. Abra este proyecto en IntelliJ IDEA.
2. Complete los métodos solicitados.
3. Use estructuras de datos.
4. Realice commits frecuentes.
5. Suba su solución con git push.

## Requisitos
- Java 18
- Maven
